# Creative menu

Адаптивная навигация Creative на чистых HTML, CSS и JS.

## Основные файлы

- `index.html` - базовая разметка хедера, модального окна и демо-контента.
- `styles.css` - все стили, состояния, адаптив и анимации.
- `script.js` - данные меню, рендеринг desktop/mobile-навигации, логика dropdown и callback-модалки.
- `assets/menu/` - логотип, иконки, изображения модулей и наград.

## Где редактировать меню

Основной источник данных меню находится в `script.js` в массиве `navTree`.

Один и тот же `navTree` используется для desktop dropdown и мобильной пошаговой навигации. Если добавить, удалить или переименовать пункт в `navTree`, изменение автоматически попадет в обе версии.

## Где редактировать визуальные параметры

В `styles.css` есть секции:

- `Design tokens and base page styles` - цвета, базовые значения, фон страницы.
- `Header navigation` - шапка, логотип, пункты меню, телефон, кнопки.
- `Desktop dropdown` - desktop-выпадающие списки.
- `Shared overlay and callback modal` - затемнение, callback-модалка, форма, success-состояние.
- `Responsive rules` - брейкпойнты и мобильные стили.

## Брейкпойнты

- `1300px` - скрывается телефон в desktop-хедере.
- `1068px` - desktop-меню переходит на вторую строку.
- `694px` - включается мобильная навигация.
- `479px` - callback-модалка становится полноширинной.

## Важно

- Отправка callback-формы сейчас имитируется на фронтенде. Backend-интеграция намеренно не добавлялась.
- Не дублируйте меню вручную в HTML. Новые пункты нужно добавлять в `navTree`.
- Если данные меню когда-нибудь будут приходить из CMS/API, оставьте экранирование через `escapeHTML()` в рендер-функциях.
